## GO语言常量的声明

```go
var const type func


例：
package main

import "fmt"

const boilingF = 212.0

func main() {
    var f = boilingF
    var c = (f - 32) * 5 / 9
    fmt.Printf("boiling point = %g°F or %g°C\n", f, c)
    // Output:
    // boiling point = 212°F or 100°C
}


在包一级声明常量
const boilingF = 212.0   可在整个包对应的每个源文件中访问

f,c是函数内部的常量   能在函数内部很小的范围被访问






init()函数
每个文件都默认包含一个init函数
```
