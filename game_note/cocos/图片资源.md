Premultiply Alpha
勾选之后会将 RGB 通道预先乘以 Alpha 通道

1.勾选
    比如透明度为 50% 的红色，RGB 为（255, 0, 0），预乘之后存储的颜色值为（127，0，0，0.5）。

2.非勾线
    那么上面所述的透明度为 50% 的红色，存储的颜色值则为（255，0，0，0.5）




在什么场景下是需要勾选 Premultiply Alpha的
    在图形渲染中透明图像通过 Alpha Blending 进行颜色混合


原
    result = source.RGB * source.A + dest.RGB * (1 - source.A)
    gl.blendFunc(gl.SRC_ALPHA, gl.ONE_MINUS_SRC_ALPHA)


勾选  Premultiply Alpha

result = source.RGB + dest.RGB * (1 - source.A);
gl.blendFunc(gl.ONE, gl.ONE_MINUS_SRC_ALPHA)
