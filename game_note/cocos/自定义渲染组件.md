## 自定渲染
![自定渲染](./img/render-component.png)
```js
cc.RenderComponent 所有的直接渲染组件直接继承该类，如  cc.Sprite、cc.Label、cc.Graphics



例 
   const assembler = Label.Assembler!.getAssembler(this);

   render2d 类
   public static Assembler: IAssemblerManager | null = null;



   
每个渲染组件都有其对应的 assembler 进行数据的组装和填充


Assembler 
   必须定义 updateRenderData 
      准备顶点数据
   fillBuffers  
      后者则是将准备好的顶点数据填充进 VetexBuffer 和 IndiceBuffer


      



```
