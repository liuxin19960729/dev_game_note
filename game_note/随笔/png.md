# PNG

## PNG-32
```
r,g,b,a each a channel 8bits.

```

## PNG-24

```
PNG-24 no supports aplha channel , rgb each a channel 8bits size.

```




## PNG-8
```
用到的每种颜色存储到长为255的数组上面

each piexl store array index.



由于存储的颜色种类是有上线的 所有当我们使用PNG-32 covert PNG-8 的时候会明显的感觉到颜色不是太平滑
```
