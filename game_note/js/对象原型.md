
1.原型
```
js 原型制继承方式

null 没有原型
最底层的原型为 object


js 访问 属性 方法的方式
 obj.a
 
 obj寻找-->__proto__下的对象寻找.....知道最后一层

 1.当在某一层找到返回 /执行
 2.最后一层未找到代表不存在

```

例:
```js
let f = function () {
   this.a = 1;
   this.b = 2;
}

   this.a = 1;
   this.b = 2; 这两个属性是属于 let o =new f() 里面的 o对象的


   若要原型设置使用

   f.propertype.c=100;

let o =new f()  o["__proto__"] 的原型是 f.propertype
```
