## 伴随矩阵的定理
```
任意方阵都有伴随矩阵

*****任意方阵 (A*) *A =A *(A*) =|A|E  

{
    |A| 0 0
    0   |A|0
    0  0  |a|
}


一行元素*自己的代数余子式相乘+在相加就是 行列式的值

一行元素和其他行的代数余子式相乘在相加为0


行列式 按 行展开  按 列展开都是可以的
  行 ai1j1......ainjn
  i是顺序   j逆序 偶数（-1）^(偶数) +  奇数反正

  例 j是顺序的  和上面的规则一样
```
