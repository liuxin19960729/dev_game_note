# 颜色模型


## 简介
```

人眼中有三种视锥细胞分别用来感知 红 绿 蓝的强度
人眼看到的颜色=三种不同颜色的强度组合结合人眼的生物特征





rgb 模型(add)

r+b+a=白色

r+g=黄色
r+b=品红
g+b=青色

取值范围[0-255]


cmy(k黑) 模型(青 品红  黄色)   运用在印刷 (minus)

hsv 模型
    h 色相
    s 饱和度
    v 明度 颜色亮度



hsi 模型
  h 色调
  s 饱和度
  i 亮度
  

hsl 模型

  l 明度 白色的亮度




```


### RGB Model (光学的三原色) 
### CMY(k) Model  青 品红 黄色 (印刷的三原色)
```
RGM 色料色光三原色原理

色光三原色：三种颜色自生能发光进入到眼睛产生视觉信号

色料三原色  照如一束光，光照射到一个物体 ，物体洗扫吸收一些颜色。会反

射一些颜色，反射的这些颜色被我们眼睛接收到组合就是那种颜色
 
CMK(Y)是色料三原色建立的色彩模式(减色模式:减去光的色彩)

```
