注意 在cocos里面重合就会显示混合模式
```
上层为源 src

下沉为目标 dst

opengl 是相加的计算出新的一层放在他们两的上面覆盖(新计算出来的颜色值表示在两层或多层覆盖在同一个像素上的像素上)



```

## src_alpha
```
源因子取值为源颜色的aplha
颜色向量*alpha

```

## ONE_MINUS_SRC_ALPHA
```
颜色向量*(1-源alpha)

```



例
```
dst=r 112 g 64  b 48 a 255
src=r 255 g 255 b 255 a 100


混合后
 
r=255*(100/255)+(1-(100/255))*112
   99.45+68.32
   =167.77
g= 255*(100/255)+(1-(100/255))*64
 99.45+39.04
 =138.49

b= 255*(100/255)+(1-(100/255))*48
 99.45+29.28
=128.73

```

 =255
