WebGl 提供缓冲对象(在WebGl系统中存在的一块内存区域)
可以一次性地向着色器传入多个顶点的数据

缓冲对象：
    可以一次性的传入大量的数据，将这些数据保存在webgl缓冲区里面(供顶点着色器使用)




创建WebGl缓冲区的流程
1.需要先创建缓冲区(gl.createBuffer())
    let buffer=gl.createBuffer();

    删除创建出来的缓冲区域

    gl.deleteBuffer(buffer);

2.绑定缓冲区对象(gl.bindBuffer())
    将创建出来的缓冲区绑定到webgl系统上

    gl.bindBuffer(gl.ARRAY_BUFFER,buffer)
    上面的代码表示的是我们将缓冲区绑定到顶点缓冲区上面去

    bindBuffer(target,buffer)

    target
       gl.ARRAY_BUFFER 表示缓冲区对象包含顶点数据
       gl.ELEMENT_BUFFER 表示缓冲区对象中包含了顶点索引值
       null 表示禁用buffer的绑定
    buffer就是第一步创建出来的buffer

3.将数据写入缓冲区对象(gl.bufferData())

    gl.bufferData(gl.ARRAY_BUFFER,vertices,gl.STATIC_DRAW)
    将vertices数据写到gl.ARRAY_BUFFER(顶点数组缓冲区)上面去.


    gl,bufferData(target,data,usage)

    targer  目标
    data  要写入的数据
    usage
        程序将如何使用存储在缓冲区的数据
        gl.STATIC_DRAW 只向缓冲区写入一次数据，需要绘制很多次
        gl.STREAM_DRAW 写入一次，绘制若干次
        gl.DYNAMIC_DRAW 写入多次，绘制很多从

    Floa32Array 数组WEBGL引入 保证他们同一种类型


类型化的数组
    ....


    不支持 pop  和 push方法


    支持的方法或属性

    get(index) 获取 index索引的元素值

    set (index,value)

    set (array,offset) offset开始将array数组的数据插入进去

    length() 数组的长度

    BYTES_PER_ELEMENT 数组中每个元素所占的字节数


注意 绑定数据 不能使用普通的js提供的数组对象




4.将缓冲区分配给attribute对象(gl.vertexAttributePointer())
     
    gl.vertexAttribPointer(a_position,2,gl.FLOAT,false,0,0)

    gl.vertexAttribPointer(location,size,type,normalized,stride,offset)
        location 需要分配数据的attribute的存储位置
        size :指定缓冲区每个顶点的分量个数(1-4)
            注意 size 比 attribute 修饰的 类型分量小 他会与gl.vertexAttri[1234]f()的补全规则一样的。
            例 size =1 则 y=z=0 w=1.0

        type:用于指定缓冲存储数据的格式
            gl.FLOAT Float32Array........


        normalize:bool   是否将传入的浮点数进行归一化  [0,1]  [-1,1] 之间


        stride ： 相邻两个顶点之间的字节数 default 0

        offset  :attribute 的数据从缓冲区的那个地方开始读取       

5.开启attribute属性 (gl.enableVertexAttribArrat)
    开启attribute 变量
    gl.enableVertexAttribArray(a_position) (实际处理的对象缓冲区的开启)
    使缓冲区和attribute的属性进行连接


    gl.disableVertexAttribArrat(a_position) 关闭 
    缓冲区和attribute的属性关闭






gl.drawArrays(mode,first,count)

    first 指定从那个顶点开始绘制 0 开始 相当于 缓冲区数组的开始索引

    count 绘制点的个数 (通过缓冲区依次给 attribute vec4 a_position 传入数据)




注意gl_PointSize 这个顶点做色器属性只有在绘制点的时候才有用



绘制三角形
  gl.drawArrays(gl.TRIANGLES,0,3)

  mode gl.TRIANGLES 告诉她是绘制的是三角形

  这样我们绘制的这三个点就不是相互独立是同一个三角形的点
