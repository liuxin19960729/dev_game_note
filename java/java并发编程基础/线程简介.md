```
现代的操作系统基本采用的时分的调度运行线程
线程优先级就是决定线程需要多或者少分配一些处理器资源线程属性

java线程中通过整形的 priority 属性控制优先级 [1,10] ,default priority ==5


高优先级分配时间多于低优先级


设置线程优先级时:针对频繁阻塞(休眠，或者 io) 线程需要这只较高的优先级，偏重运算设置较低优先级，确保处理器不会独占。

在不同的的jvm和操作系统上线程的规划存在差异





```


### 线程的状态
```
new  线程被构建 未调用 start() 方法

running 操作系统的就绪个运行两种状态叫运行中


bloked 阻塞

waiting 等待状态，运行需要其他线程的通知或中断


time_waiting 可以指定时间

terminated 终止状态 表示当前线程已经执行完毕


```

### Daemon线程

```
daemon 线程主要被用于程序中后台调度及支持性的工作

线程的daemon 设置只能在调用start()方法之前设置




程序随着mian线程(主线程 非deamond)的终止而终止


jvm 会随着没有 非daemon线程的退出而溢出


所有finally 在daemon里面可能不会执行

```
