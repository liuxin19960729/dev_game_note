
### 多选按钮 和 单选
```
注意:在多选按钮中允许多个按钮同时选中,对同一个按钮 连续点击

一定是 选中--未选中.....的状态


注意 
    up 和 over 一定是一组的

        未选中 鼠标未在按钮上  up 
        未选中 鼠标在按钮上  over


    down 和 selectedOver 一定是一组的
        
        和上面相反
            表示已经选中  鼠标未在按钮上面 down
            表示已经选中  鼠标在按钮上面 selectedOver


```


### 普通按钮
```
up 未点击

down 鼠标按下

over  鼠标在按钮上

selectedOver (没有用)
```
