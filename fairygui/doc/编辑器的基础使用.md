## 编辑器的使用小技巧
#### 一.资源导入的方式
![导入资源的方式](./img/组件/导入资源的方式.png)
```
两种方式
1.点击创建的包-->点击上图1按钮--然后根据按钮选择要导入的资源


2.直接要导入的资源放入对应的包里面-->然后点击2按钮刷新
```


#### 二.资源的粘贴
![资源的粘贴](./img/组件/资源的粘贴.png)
```
1.重命名
    遇到同名重新命名
2.替换
    覆盖同名的资源

3.跳过
    test1 包里面的A组件 使用 a.png 图片  向test2包里面进行粘贴资源 但是test2 里面有 a.png 资源若选择[【跳过】则复制到test2包里面的A组件  使用的 a.png图片 就会使原来 test2包里面的a.png图片           
```
