## 窗口系统
```
FairyGUI使用约定名称将一些常见的窗口功能和我们定义的组件关联起来



下面的约定是可选,是否含有组件frame，或者组件frame里是否含有约定的功能组件，并不会影响窗口的正常显示和关闭。

frame 组件 ：作为窗口的背景
    closeButton 窗口的关闭按钮


```


### 使用窗口
```js
 Window win = new Window();
win.contentPane = UIPackage.CreateObject("包名", "内容组件名").asCom;
win.Show();



```

show()方法执行流程
![窗口Show流程](./img/窗口Show流程.png)

```
onInit() 首次执行 (是否初始化)

doShowAnimation() 
    如果你需要【窗口显示时播放动画效果】，那么覆盖DoShowAnimation编写你的动画代码，并且在动画结束后调用onShown()


OnShown() 业务逻辑

```

hide()方法执行流程
![隐藏hiden流程](./img/隐藏hiden流程.png)

```
doHideAnimation()
    隐藏页面的动画

OnHide()
    编写其他需要在窗口隐藏时处理的业务逻辑

```
