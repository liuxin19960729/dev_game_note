## 极限定义
```
极限定义公式
｜xn-a｜< ξ

a表示极限

这个公式的可以解释
ξ 半径
以a为极限

只要 xn 只要 xn落在 半径里面就满足

XN 其实表示的是 
｜xn-a｜=ξ 半径上的那个数

当 当n >N 对应当数满足 ｜xn-a｜< ξ
那么当N 扩小时 依然满足 n>N  ξ不变那坑定还是满足 ｜xn-a｜=ξ 因为 n 当范围依然在以前那个范围里面
```

## 性质
```
1.数列收敛，极限唯一

2.数列收敛，元素极限有界
  ｜xn-a｜<ξ 
  当 假设 ξ =1 ，存在N n>N  满足 ｜xn-a｜<ξ
   xn=｜xn-a+a｜=|xn-a|+a
   1+a>xn
  通过定义｜xn-a｜=ξ  只能求出 n>N当界限
  n>N  界限 xn <｜a｜+1
  v=max(x1,x2...,|a|+1)


3.有界是收敛当必要条件
    不是充分条件
    利  {1,-1,1,-1....-1} 有界 但是不是收敛
    ** 单调性有界 则有极限。
4.极限 a  a>0(a<0) 存在N n>N xn>0
  a>0      ξ=a/2
  n>N   
  |xn-a|<2/a
   - a/2<xn-a<a/2
   a/2<xn<3a/2
   a/2>0
   所以存在 xn>0

5.子数列
   存在数列  x1  x2 .....x6...xn

   抽取子数列 x2 x4.....xn-1  (保证抽取出来的数列保持顺序不改变)

    kn>n

6.数列{xn}的极限为 a 那么它的任何子数列{xkn} 的极限也为 a。

证明  
 {xn} 极限 为a 存在N n>N   |xn-a|<ξ 
 由于上面的性质克制
 kn>=n>N
 所以满足极限定义的式子。｜x(kn)-a|<ξ 



   一个极限，能找到一个子数列不收敛，原数列就不收敛

   遭到两个子数列收敛 ，极限不同 欲拿来的数列不收收敛

   例：
      {-1 ,-1,1,-1,1....} 

      子数列1{1,1,1....1} 收敛 极限1

      子数列2{-1,-1,-1...-1} 收敛 极限-1

      子数列极限不同，虽然都收敛，它还是不是收敛都数列

   原数列《==》奇数项子数列和偶数项子数列都是收敛都并且极限相同  就能推出原数列收敛
```
