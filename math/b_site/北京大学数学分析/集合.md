```
1.奇偶性
   奇函数
   f(-x)=-f(x)  fx=-f(-x);  x 支持所有 X(定义域)
   
   偶函数 f(x)==f(-x)  x 支持所有 X(定义域)


函数存在奇偶性的前提  定义域必须要是 x为 0 轴 对称的


单调性

    对于任意 x1 x2 属于定义域X,且 x1>x2
    f(x1)>=f(x2) 单调递增 若> 是严格单调上升

    对于任意 x1 x2 属于定义域X,且 x1<x2
    f(x1)<=f(x2) 单调递减
```
